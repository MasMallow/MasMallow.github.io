# Albion Comp Manager

เว็บจัดเซ็ตอุปกรณ์ + จัดคอมป์ทีม 5 / 10 / 20 คน สำหรับ Albion Online

🔗 **เปิดใช้งาน: https://masmallow.github.io/**

- สมาชิกกิลด์: เปิดลิงก์ดูคอมป์ได้เลย ไม่ต้องล็อกอิน (อัปเดตเรียลไทม์)
- ผู้จัดทีม: กดปุ่ม 🔑 **ผู้จัดทีม** มุมขวาบนเพื่อล็อกอินแล้วแก้ไขได้
- ข้อมูลเก็บบน [Supabase](https://supabase.com) — ตั้งค่าด้วยไฟล์ `supabase-setup.sql` แล้วใส่คีย์ใน `config.js`

> โค้ดต้นทางและคู่มือ deploy อยู่ในเครื่องผู้ดูแล (โฟลเดอร์ `albion` — ดู `DEPLOY.md`)
> อัปเดตเว็บนี้ด้วยการรัน `publish.bat` จากเครื่องผู้ดูแล
